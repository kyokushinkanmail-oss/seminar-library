はじめに ― このセミナーの目的

このセミナーでは 基本から考えていく。空手を始めて一番最初に習う基本稽古。これが実際の組手ではどう使われているのか、また基本の考えをどのように応用していくのかを知ることは、そのまま 武器 になる。

近距離で無茶苦茶効果を発揮する。力を伝えやすい「ほぼタックル」、超絶高火力 ─ 特に肘。
これが今回のテーマ：肘と膝。

第1章　なぜ今、肘と膝なのか

距離のある組手は当然必要 ─ ただし

現代の組手では、距離をしっかり取った戦いができることはもちろん大前提となっている。しかし、どれだけ距離を意識していても 避けられないのが近距離戦 である。

その理由はシンプルで、ほとんどの選手は組手の時 「前に出たい」 と思っているからだ。

なぜみんな前に進みたいのか

前に進んでいる方が「有利に見える」から。

みんなの本音

  • 試合に勝ちたい
  • 有利に見られたい → 前に進みたい
  • お互いにそう思っているので、結果として極端に距離が近くなる

つまり、組手では 必ず近距離戦が起きる。これは構造上の必然である。

第2章　近距離戦の構造

結論

  1. 接近戦になった時に、相手が「そこから離れたい」と思う状況を作る
  2. その結果、相手が下がる ＆ 距離を作れる
  3. 判定にも有利に影響する
  4. その武器となるのが 膝・肘

近距離になった時に 相手が「ここに居たくない」と感じる ─ この状態を作れるかどうかが、近距離戦の主導権を分ける。

第3章　距離の理解

近づく方が、距離を取るより簡単

距離を取ろうとして下がるよりも、相手に詰めて近距離に入る方が物理的にもタイミング的にも難しくない。だから接近は起こる。

正拳突きのベスト距離は、思っているより遠い

正拳突きが最も力を発揮する距離は、多くの人が想像しているより少し遠い位置にある。一方、組手における近距離は 手が伸び切る距離より、さらに近いことがほとんど である。

この近距離帯では、突きや蹴りなど射程の長い攻撃は十分に伸びきれず、力を発揮しづらい。
→ 射程の短い攻撃＝肘・膝の方が力を発揮できる

第4章　肘 ― 殺傷能力の塊

肘の基本

  • 肘頭（ちゅうとう）で当てる ─ 肘の尖った骨で打つ
  • 基本的には 膻中（だんちゅう）や胸を狙う
  • 肘頭の尖った所で当てると、死ぬほど痛い（たまに死ぬ ─ それくらいの破壊力）
練習でも実戦でも扱いには細心の注意を。威力が高いぶん、コントロールできて初めて武器になる。

肘頭を当てる方法

正しく当てるためのキー：肘を出す時に、手を内側に回す。これで自然と肘頭が前に出る。

練習方法

  • 地面に肘頭を付ける（感覚づくり）
  • ミットにくっ付ける
  • 徐々に叩いていく

そして何より重要 ─ ポジションが無茶苦茶大事。

当てる条件と方法

膻中や胸を当てに行く時の条件：

  • 相四つ（あいよつ）必須
  • 肘は拳に比べて 関節が少なく、角度の調整がしづらい
  • そのため 相手と自分の位置関係 が決定的に重要になる

当てるパターン解説 ─ 角度と相手の奥手

  • 角度 ─ 相手の胴体の正面から肘を打つ
  • 奥手は邪魔になる ─ そのままでは奥手にブロックされる
  • 対策 ─ 他の攻撃を受けさせて、相手の奥手を「外から受けるかたち」 に誘導する

つまり肘は単発で当てに行くのではなく、相手の手の動きと位置を作ってから刺すイメージ。

第5章　膝 ― 近距離最強の蹴り

膝は 勝敗が変わるくらい重要。近距離最強の蹴り、ほぼタックル。

当てる場所

  • 膝の内側の上
  • 脛骨（けいこつ）の一番上

膝の当て方

① 膝の内側の上で当てるパターン

  • 奥足側を 喧嘩四つ にして、相手の胴体に対して真っ直ぐ蹴る
  • 相四つからそのまま蹴ってもいい

② 脛骨の一番上で当てるパターン

  • 基本の前蹴りのような感覚で、腰を前に出して蹴る
  • このフォームなら 好きなタイミングで蹴れる ─ ここが大きな強み
※フォローまで繋げる意識で。膝が当たって終わりではなく、その後の展開を作る一手として。

第6章　肘・膝から距離を作る

肘や膝が入ると 高火力 なので、相手は痛みと圧で 少しスペースを作ってくれる。これが連携のチャンス。

使えるパターン例

肘 → 中段回し蹴り（肘で押し下げて、空いた中段を蹴り抜く）

膝 → 前蹴り（膝で距離を作り、開いたスペースを前蹴りで支配）

膝 → 肘（高火力を二段重ねで畳み掛ける）

まとめ

  1. 近距離は組手中、絶対に起こりうる距離
  2. 距離を取るためにこそ、高火力な攻撃を持っておく
  3. 近距離になった時に 「ここに居たくない」と思わせる
  4. 結果、相手が後退する → 判定で大きく有利
  5. 肘打ち・膝蹴りを練習しましょう！！
